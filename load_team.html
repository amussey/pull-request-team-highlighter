What do I put in the box?

You should paste into this box a JSON blob. This blob should have three elements: The name of your team (name), a list of your team members (members), and the color you want to use to denote that team as either a hex string or an rgb/rgba string (color) Here is an example: 
[
    {
        "name":"YOLO",
        "members":["amussey","thegreenrobot"],
        "color":"rgba(255, 255, 0, 0.37)"
    },
    {
        "name":"Blue",
        "members":["will","bill"],
        "color":"#0000FF"
    }
]
